What is this page?

This is a collection of 20 fun, simulated datasets designed for teaching introductory and intermediate statistics courses using R. Instead of boring, generic data, students get to work with Star Wars characters, Hogwarts grades, zombie survival scenarios, and more — all while learning real statistical concepts like regression, ANOVA, chi-square tests, and data visualization.

Each dataset is a CSV file ready to download and load straight into R. Sample sizes range from 75 to 2,000 observations with realistic variable distributions, including intentional missing data for teaching opportunities. Click any dataset below to download it!

:: Practice R Scripts ::

Each dataset above has a corresponding R script designed for 10-15 minute instructional walkthroughs. Scripts 1-10 are for a PhD-level Intro to Data Analysis course. Scripts 11-20 are for a PhD-level Advanced Statistics course covering multiple regression, ANOVA, logistic regression, factor analysis, and more.
